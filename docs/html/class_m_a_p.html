Detailed Description

Represents a grid-based map.

Author
Hayri Baran Vural (bgnvr.nosp@m.l@gm.nosp@m.ail.c.nosp@m.om)
Date
December 2024 The MAP class provides functionality for managing a grid-based map, allowing points to be inserted, cleared, and visualized.
